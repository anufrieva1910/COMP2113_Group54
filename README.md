# Game Description

You play as a wandering Cat Warrior, descending into a cursed cave to face the all-mighty Cerberus, a three-headed beast said to haunt its deepest chamber. As you go deeper, you find ancient drawings carved into the stone; paintings that tell the story of how the beast came to be. 

Each room features a room, where you explore the surroundings, fight the enemies, solve ancient puzzles and discover hidden passagways. Some rooms uncover the secrets of this place, revealing the truth through found lore fragments. Challenge yourself by facing the Cerberus in a three-phase boss fight on the last floor.

The game has three difficulty modes: Easy, Normal and Hard. Hard mode features a hardcore gameplay, featuring permadeath - no saving.
